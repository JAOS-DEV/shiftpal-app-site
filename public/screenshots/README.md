## Usage

### In React Components

```tsx
import Image from 'next/image';

<Image
  src="/screenshots/tracker-manual.png"
  alt="ShiftPal tracker screen showing manual shift entry"
  width={414}
  height={896}
  className="rounded-lg shadow-xl"
  priority // for hero images
/>
```

### Optimization

- Use Next.js Image component for automatic optimization
- WebP format automatically generated by Next.js
- Lazy load screenshots below the fold
- Add proper alt text for accessibility

## File Naming Convention

Keep filenames lowercase with hyphens:
- ✅ `tracker-manual.png`
- ❌ `TrackerManual.png`
- ❌ `tracker_manual.png`

## Image Specifications

- **Format:** PNG (Next.js will optimize)
- **Device:** iPhone screenshots
- **Aspect Ratio:** ~19.5:9 (standard iPhone)
- **Recommended Width:** 1242px (3x) or 828px (2x)
- **Display Width:** 414px (1x) scaled appropriately
